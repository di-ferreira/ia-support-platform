# Projeto

Sistema de Atendimento WhatsApp com IA para ERP Autopeças

## Objetivo

Automatizar suporte técnico através de:

- WhatsApp
- Evolution API
- N8N
- IA
- RAG

## Fluxos

### Cenário A

IA resolve diretamente.

### Cenário B

IA encontra solução.

Humano executa.

### Cenário C

IA não encontra solução.

Humano investiga.
